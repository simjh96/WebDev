	Access-Control-Allow-Origin
		cors(cross origin resource): CORS or cross origin resource sharing is blocked (in javascript APIs) in modern browsers by default
			두가지가 만족 되어야 이런일이 안생김
				1. response 의 Access-Control-Allow-Origin 이 내 request ip와 같을것
				2. 해당 ip가 localhost 가 아닐것
					이런 경우 preflight response의 Access-Control-Allow-Origin 를 * 한줄을 더 추가하는 plugin을 설치함으로 cors 보안 회피

	axios는 res를 .data에 다 넣어놓음
